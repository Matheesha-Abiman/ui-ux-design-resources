# 🎨 UI vs UX  

## 🖌️ UI (User Interface)  
The **visual layer** of a product.  

Includes:  
- Buttons  
- Colors  
- Typography  
- Layout  
- Icons  

👉 Focus: *Look & Feel*  

---

## 🧠 UX (User Experience)  
The **overall user journey and usability**.  

Includes:  
- Navigation  
- User flow  
- Task completion  
- Interaction  

👉 Focus: *Ease of Use*  

---

# ⚡ UX Laws (IMPORTANT)  

### Hick’s Law  
More choices = slower decisions  

👉 Keep UI simple  

---

### Fitts’s Law  
Bigger & closer elements are easier to click  

👉 Use large buttons  

---

### Jakob’s Law  
Users expect familiar patterns  

👉 Don’t reinvent UI  

---

### Law of Prägnanz  
Users prefer simple designs  

👉 Avoid complexity  

---

# 📱 Responsive Design  

## Breakpoints  

| Device | Width |
|------|------|
| Mobile | 320–480px |
| Tablet | 481–768px |
| Laptop | 769–1024px |
| Desktop | 1025px+ |

---

## Constraints  

- Left / Right → Fixed position  
- Top / Bottom → Vertical alignment  
- Center → Always centered  
- Stretch → Responsive resizing
